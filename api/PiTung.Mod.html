FrameworkVersion

The version of PiTUNG this mod is using. If a version number is not included, it is considered a wild card. E.g, if this property equals "1.0", the mod will work on PiTUNG version "1.0.123", but not if this property equals "1.0.0".

Setting this property to FrameworkVersion is strongly discouraged. If you need your mod to be able to be loaded in any PiTUNG version, set RequireFrameworkVersion to true.
